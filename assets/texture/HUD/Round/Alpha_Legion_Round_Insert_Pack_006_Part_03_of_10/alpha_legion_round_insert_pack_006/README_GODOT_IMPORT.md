# Godot Import Notes

Use each PNG as a single circular insert under your octagonal frame.

Recommended Godot 4.x import settings:

- Keep alpha enabled.
- For crisp HUD work, use nearest/linear according to the final scale of your UI.
- Disable repeat unless you deliberately tile the insert.
- If the insert is scaled far below source size, enable mipmaps to reduce shimmer.
- Put the octagonal frame above the insert in draw order.

The center markings are intentionally abstract and non-official; they are meant to carry grimdark cogitator/noosphere mood without copying protected marks.
